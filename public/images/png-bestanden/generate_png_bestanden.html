Stap 2: Download de bestanden

Nadat je op "Genereer PNG Bestanden" hebt geklikt, verschijnen hier downloadlinks. Klik op elke link om het bestand te downloaden en sla het op in deze map.

Stap 3: Gebruik de PNG bestanden

Na het downloaden kun je de PNG-bestanden direct gebruiken in je presentaties, documenten of andere toepassingen.

Voorbeeld HTML:

<img src="/images/png-bestanden/buurtconnect-logo-300x100.png" alt="BuurtConnect" />

Voorbeeld React:

import Logo from '/images/png-bestanden/buurtconnect-logo-300x100.png';

function Header() {
  return <img src={Logo} alt="BuurtConnect" />;
}

Gebruik op Vercel

Voor gebruik op Vercel hosting:

  1. Zorg dat de PNG-bestanden in de public/images/png-bestanden/ map staan
  2. Gebruik absolute paden vanaf de root: /images/png-bestanden/buurtconnect-logo-300x100.png
  3. Bij gebruik van Next.js, overweeg het Next/Image component te gebruiken voor optimalisatie

Zie vercel-info.md in deze map voor meer details en probleemoplossing.
